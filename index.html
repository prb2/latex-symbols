Latex Symbol Search

A Flashlight plugin for finding LaTeX symbol syntax

View on GitHub Download .zip Download .tar.gz

A plugin for the Mac app Flashlight that allows you to search for symbols and find their LaTeX command.

Installation

If you do not have Flashlight install, follow the instructions here.

In the Flashlight app, search for Latex Symbol Search and install the plugin.

Usage

The plugin can be triggered by typing either tex or latex into the Spotlight search bar. Follow the keyword with the symbol you wish to search for.

tex pi
latex arrow
